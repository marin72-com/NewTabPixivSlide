NewTabPixivSlide
================
新しいタブを開く度にpixivの画像をスライドするchromeの拡張機能です.
pixivのログイン時: ブックマーク画像(ない場合はデイリーランキング画像)
pixivの非ログイン時: デイリーランキング画像
を表示します.


# 使用方法
----
1. Zipをダウンロードし解凍します.

2. chromeを立ち上げ，chrome://extensions/ をアドレスバーに入力します.

3. "デベロッパーモード"にチェックを付け，"パッケージ化されていない拡張機能を読み込む"をクリックしてください.

4. 先ほど解凍したフォルダを指定します.
